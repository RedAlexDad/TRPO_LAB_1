 Этот проект реализует систему автоматического распознавания речи с использованием API Yandex Speech\+Kit. Основной класс {\ttfamily \doxylink{namespaceAutomaticSpeechRecognitionYandex}{Automatic\+Speech\+Recognition\+Yandex}} предоставляет функциональность для обработки аудиофайлов, определения их формата и отправки запросов на распознавание речи. Производный класс {\ttfamily \doxylink{namespaceAdvancedSpeechRecognition}{Advanced\+Speech\+Recognition}} расширяет базовый класс, добавляя возможность логирования результатов.\hypertarget{index_autotoc_md1}{}\doxysection{\texorpdfstring{Основные возможности}{Основные возможности}}\label{index_autotoc_md1}

\begin{DoxyItemize}
\item Определение формата аудиофайлов (WAV, MP3, OGG).
\item Конвертация аудио в формат Ogg\+Opus.
\item Распознавание речи через API Yandex Speech\+Kit.
\item Логирование результатов в производном классе.
\end{DoxyItemize}
